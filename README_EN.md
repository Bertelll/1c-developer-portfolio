# 🚀 Developer OS

> A production-grade operating system for software developers.

Developer OS is a personal ecosystem that combines portfolio, knowledge base, project management, AI assistance, career tracking and continuous learning into one modern application.

The goal of this project is not only to build software, but to demonstrate production-level engineering practices.

---

# ✨ Features

- 📁 Portfolio Management
- 📚 Knowledge Base
- 📝 Notes
- ✅ Task Manager
- 🤖 AI Assistant
- 💼 Career Dashboard
- 📊 GitHub Analytics
- 🎯 Learning Tracker
- 📈 Progress Metrics
- 🌙 Dark / Light Theme
- 🌍 Multi-language support
- 🔐 Authentication
- ⚡ Responsive UI

# 🚀 Getting Started

Clone repository

```bash
git clone https://github.com/Bertelll/Developer-OS.git
```

Frontend

```bash
cd frontend

npm install

npm run dev
```

Backend

```bash
cd backend

python -m venv .venv

pip install -r requirements.txt

uvicorn app.main:app --reload
```
